# 🧮 Tkinter_Calculator 🤓
Welcome to Khushboo's Calculator! 🎉 A simple yet powerful calculator built with Python and Tkinter. 💻✨

# 📜 Features
Basic Operations: Addition ➕, Subtraction ➖, Multiplication ✖️, Division ➗

Advanced Functions: Percentage 📉, Exponentiation 📈

Clear Button: Reset your input with "C" ❌

Decimal Support: Use decimal points with "." 🔢

Error Handling: Display errors gracefully 🚨

# 🚀 Getting Started
Prerequisites

Python 3.x 🐍

Tkinter library (usually included with Python) 📦
# 💡 How to Use
Enter Numbers: Click the numeric buttons or type directly on your keyboard. 🔢

Perform Calculations: Use the operation buttons to perform calculations. 🧮

Get Results: Click "=" to see the result of your calculations. 🆗

Clear Input: Use "C" to clear the input field. ❌
